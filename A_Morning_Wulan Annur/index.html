About

Nama saya Wulan Annur, seorang mahasiswa berusia 21 tahun yang lahir di Tasikmalaya pada 19 Oktober 2002. Saat ini, saya menempuh pendidikan di Universitas Galuh, jurusan Manajemen. Sejak masa sekolah menengah kejuruan (SMK), saya telah memiliki minat dan menguasai bidang bisnis daring dan pemasaran. Selain itu, saya juga tertarik dengan bernyanyi, berbisnis, dan berbicara di depan umum. Sejak kecil, saya telah terbiasa untuk mandiri dan bekerja keras. Saya dibesarkan oleh keluarga yang sederhana, tetapi orang tua saya selalu mengajarkan saya untuk tidak menyerah dan terus berusaha mencapai impian saya. Saya percaya bahwa setiap orang memiliki potensi untuk sukses, asalkan mereka mau bekerja keras dan tidak takut untuk gagal. Selama masa sekolah menengah kejuruan, saya aktif mengikuti kegiatan ekstrakurikuler di bidang kewirausahaan. Saya juga mengikuti beberapa kompetisi bisnis daring dan berhasil meraih beberapa penghargaan. Pengalaman ini telah membantu saya untuk mengembangkan keterampilan saya dalam bidang bisnis daring dan pemasaran. Setelah lulus dari sekolah menengah kejuruan, saya memutuskan untuk melanjutkan pendidikan saya di Universitas Galuh. Saya memilih jurusan Manajemen karena saya ingin memiliki pengetahuan dan keterampilan yang lebih mendalam di bidang bisnis. Saya juga berharap bahwa dengan pendidikan yang saya jalani, saya dapat mengembangkan bisnis daring saya sendiri di masa depan. Saya berkomitmen untuk terus belajar dan berkembang. Saya juga ingin menggabungkan pengetahuan dan pengalaman saya untuk mencapai tujuan karier saya dan memberikan kontribusi positif pada industri. Saya percaya bahwa dengan kerja keras dan dedikasi, saya dapat mencapai impian saya untuk menjadi seorang pengusaha yang sukses.
